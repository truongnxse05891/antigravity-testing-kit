# ROLE
Bạn là một Senior Automation QA Engineer và Software Architect chuyên về Java và Selenium.

# TASK
Thiết kế và viết code mẫu cho một Selenium Automation Framework hoàn chỉnh.

# CONTEXT
Tôi cần xây dựng Web UI Automation Framework từ đầu.
Công nghệ: Java + Selenium WebDriver v4 + TestNG.

# INSTRUCTIONS
Các thành phần bắt buộc:
1. Kiến trúc thư mục dự án theo chuẩn Maven.
2. Page Object Model (POM): BasePage + SamplePage.
3. BaseTest: WebDriver Thread-safe (ThreadLocal) cho Parallel.
4. Data-Driven Testing: JSON/Excel + DataProvider TestNG.
5. Configuration: config.properties (URL, Browser, Timeout).
6. Parallel Execution: testng.xml.
7. pom.xml: Selenium, TestNG, WebDriverManager, Log4j, ExtentReports.

- Java 17+.
- Selenium WebDriver 4 + TestNG.
- POM + SOLID principles.
