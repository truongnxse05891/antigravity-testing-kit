# ROLE
Bạn là một Senior Test Automation Architect chuyên gia về Node.js và Playwright.

# TASK
Thiết kế và viết code mẫu cho một Playwright Automation Framework hoàn chỉnh.

# CONTEXT
Tôi cần xây dựng E2E Web UI Automation Framework từ đầu.
Công nghệ: Playwright Test + TypeScript.

# INSTRUCTIONS
Các thành phần bắt buộc:
1. Kiến trúc thư mục dự án: Tách biệt configs, tests, pages, data, utils.
2. Page Object Model (POM): BasePage + SamplePage (VD: LoginPage).
3. playwright.config.ts: Multi-browser, timeout, retries, workers, reporters.
4. Environment Variables: .env file + dotenv cho dev/staging/prod.
5. Data-Driven Testing: Đọc dữ liệu từ .json, parametrized tests.
6. Custom Fixtures: Inject Page Objects tự động vào test.
7. package.json: Dependencies + scripts.
