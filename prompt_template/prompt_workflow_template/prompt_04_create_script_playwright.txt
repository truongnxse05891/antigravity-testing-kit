# TASK
Viết automation test script từ test scenario bằng Playwright.

# CONTEXT
- Language: TypeScript
- Test Framework: Playwright Test
- Design Pattern: Page Object Model (POM)
- Build tool: npm

Test Scenario:
[DÁN TEST SCENARIO VÀO ĐÂY]
URL: [URL hệ thống]
Tài khoản test: [EMAIL] / [PASSWORD]

# INSTRUCTIONS
1. Sử dụng Playwright MCP để mở URL và inspect DOM thật. TUYỆT ĐỐI không đoán locator.
2. Áp dụng POM — tách Page class và Test class.
3. Ưu tiên locator semantic:
   getByRole() > getByLabel() > getByPlaceholder() > getByText() > getByTestId() > CSS
4. Auto-waiting:
   Dùng: await expect(locator).toBeVisible()
   KHÔNG dùng: page.waitForTimeout()
5. Test Data unique: auto_[testName]_[timestamp]@test.com
6. Code clean, comment tiếng Việt. Pattern: Arrange > Act > Assert
7. Sau khi viết xong, tự chạy test. Nếu FAIL thì tự sửa đến khi PASS.

# OUTPUT FORMAT
1. Page Object: [Tên]Page.ts
2. Test file: [tên].spec.ts
3. Utils: DataGenerator.ts (nếu cần)
